Mi egy teszt?

Egy olyan program, ami egy másik program működését ellenőrzi.

A tesztelésről

Mire jó a tesztelés?

  • Egyes kódrészek megfelelően futnak le - unit
  • Több kódrész megfelelően illeszkedik egymáshoz - integration
  • Az alkalmazás kimenete az, amit szeretnénk - end2end/functional
  • a tesztek fajtái is itt vannak
  • unit tests - erről lesz leginkább szó
  • integration tests
  • end2end/functional tests
  • mindben más eszközök segítenek, más szemléletet igényelnek
